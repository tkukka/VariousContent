# QuickApp

Speedgauge simulation with Qt Quick.

## Development Environment

* Windows 10
* Qt 6.2.4  (open source license)
  * MinGW 11.x
  * Qt Quick 3D
  * Qt 5 Compability Module
  * Qt Shader Tools
  * Qt Multimedia (for audio dev)
  * Qt Remote Objects (for RO dev)
  * Qt Virtual Keyboard
  * Qt Debug Information Files
  * Qt Quick Timeline
* Qt Creator 7.0.1
